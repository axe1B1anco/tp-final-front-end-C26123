# Tienda de Autos Antiguos 🚗

Proyecto final del curso de JavaScript, HTML y CSS (TalentoTech). Tienda online de autos con estilo Rat Rod / Hot Rod, con catálogo de productos y carrito de compras funcional.

## Funcionalidades

- Catálogo de productos cargado dinámicamente desde JSON
- Carrito de compras (agregar, eliminar y vaciar productos)
- Persistencia del carrito con `localStorage`
- Contador de productos en el header
- Formulario de contacto

## Tecnologías

- HTML5
- CSS3
- JavaScript (Vanilla, ES Modules)

## Cómo verlo

Cloná el repositorio y abrí `index.html` en el navegador:

```bash
git clone https://github.com/axe1B1anco/tp-final-front-end-C26123.git
```
